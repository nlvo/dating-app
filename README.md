# Dating-app
## Installation
1. `$ git clone https://github.com/nlvo/dating-app.git`
2. Run  `$ npm install`
3. Run  `$ npm run start` 

## Required 
* [case](https://www.npmjs.com/package/case)
* [imagemin](https://www.npmjs.com/package/imagemin)
* [validator](https://www.npmjs.com/package/validator)
